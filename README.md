### 💫 Overview

대형 게임 타이틀에서 요구되는 **고성능 그래픽스 스택**과 **실시간 네트워크 최적화**에 깊은 관심을 두고 있습니다.  
유저에게 끊김 없는 고차원의 경험을 선사하기 위해, 렌더링 파이프라인의 효율성과 타이트한 물리 동기화를 지향합니다.

*   🎯 **Core Performance**: 대규모 트래픽과 실시간 상호작용 속에서의 동기화 및 렌더링 최적화 연구
*   ⚡ **Web3D & WebRTC**: 웹 환경의 한계를 뛰어넘는 심리스(Seamless) 멀티플레이어 환경 설계
*   ⚙️ **Robust Architecture**: 확장성과 유지보수성이 검증된 인프라 및 컨테이너 기반의 서버 환경 구축
